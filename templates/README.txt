templates contains:

index.html (/)
    The main page
    ACCES: by default the first.

form.html (/form)
    This is the page where the user complete with his/her data.
    ACCES: from index page

main_form.html (/main_form)
    This is the page where the user score all audio data.
    ACCES: from form page

end_page.html (/end_page)
    This page appears when the survey is ended.
    ACCES: from main_form page

results.html (/results)
    This page has all the results from the survey matched with audio's name.
    ACCES: only if you write the extension '/results' to the index page
